## Project Setup Instructions:

First, run the development server:

```bash
npm install
# or 
yarn add
#or 
pnpm install

# Then

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

For production:

```bash
npm install
# or 
yarn add
#or 
pnpm install

# Then

npm run build
# or
yarn build
# or
pnpm build
# or
bun build

# Then 
npm run start

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
